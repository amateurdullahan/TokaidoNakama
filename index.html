Always Know Your Moves!

Tokaido Nakama automatically updates the player tunrs whenever a player complete's their actions and has a GUI showing player positions with intuitive moving based on clicking spots on the board and brings up pertinent menus based on the location chosen.

Score and Coin Tracker

Always Know the Score!

Tokaido Nakama automatically updates each players score and coins according to the locations they stop at and cards they choose. Never forget to add the points for that temple donation again!


Probability Calculator

Always Know the Risk!

Tokaido Nakama will always show the odds of what cards are left in eash deck to be drawn. Get all the info you need to decide whether it's worth being last at the Inn to stop and the Village and shop for souvenirs!


Future Features

Some features in the works are support for 2 to 5 players and an end game achievement tracker. Other planned features are Traveler powers and options to apply the variants Return Trip, Gastronomy, and Preperations. Eventually we hope to include the option to use the Crossroads expansion for the game.

About Tokaido Nakama

Inspiration

"This project was created for Holberton's capstone project to demonstrate what we have learned over the last year. My initial plan was to do a browser based version of a board game, but between Tabletopia and Board Game Arena, they've all been done. I wanted to make something new and useful. So I decided on a companion app for Tokaido. It had a very basic rules set and wonderful asthetic, and had a lot of extra features we can expand upon as we learn more and have more time to work on it. I enlisted the help of classmates Dylan Carey and Corbin Enterline, knowing they both were hobby gamers, Corbin was interested in front end and GUI coding for games, and Dylan was more capable with Python which was the language we decided to use." - Robert Rowe, Designer
